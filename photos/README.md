# Thư mục ảnh sinh nhật của bạn gái 💖

Bạn hãy thả các bức ảnh cá nhân của bạn gái vào thư mục này để hiển thị lên thiệp trái tim.

### Yêu cầu đặt tên ảnh:
- Đặt tên các ảnh lần lượt từ **`photo1.jpg`** đến **`photo16.jpg`** (Định dạng tốt nhất là `.jpg`).
- Nếu bạn có ít hơn 16 ảnh, những ô trống thiếu ảnh sẽ tự động hiển thị hình **Trái tim màu hồng phát sáng** rất dễ thương, không bị lỗi hiển thị.
